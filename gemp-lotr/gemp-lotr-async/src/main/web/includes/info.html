About

GEMP is a general-purpose game engine focused on trading card games. There are currently only two instances of GEMP, one for the Lord of the Rings Trading Card Game (which is the one you're on), and another for the Star Wars CCG here.

GEMP was originally built to emulate Legend of the Five Rings, which is where the name comes from. Gempukku, meaning a coming-of-age ceremony in Japanese, was intended to mirror the famous Apprentice platform for Magic: the Gathering.

MarcinS was the original creator, who started up the project in August 2011. It was soon after released in beta, and soon overtook other digital platforms as the premier experience for the Lord of the Rings TCG.

In 2021, the Player's Council took over maintenance of GEMP and soon afterwards took over operation as it was migrated to PC servers, where it runs to this day. Ketura is the current head maintainer.

GEMP is written with a Java and MySQL back-end and a frameworkless Javascript on the front. It runs in a single Docker container (tho in the future we hope to expand that to multiple parallel containers). You can find the code on GitHub here.
